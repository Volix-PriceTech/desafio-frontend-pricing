# Desafio Técnico Frontend — Painel de Margem do Catálogo

Este repositório faz parte do processo seletivo da Volix. O enunciado completo está em: https://volix-pricetech.github.io/desafio-frontend-pricing/

## Setup

```bash
npm install
npm run dev    # abre em http://localhost:3000
npm test       # roda a suíte de validação
```

## Estrutura

- `src/data/catalogo.json` — dados do catálogo exportados do ERP
- `src/lib/pricing.js` — funções de cálculo (implemente aqui)
- `src/lib/pricing.test.js` — testes de validação
- `public/img.png` — referência de layout (não obrigatório seguir)

Recharts já vem instalado, mas você pode trocar por outra lib se preferir.
